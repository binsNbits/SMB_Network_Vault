# Project Pricing

Non-retainer or pre-retainer work. Projects are where margin lives; the boundary between retainer work and a project is defined in [[Retainer Scope vs Project Scope]].

| Project | Price | How-to |
|---|---|---|
| [[Security Snapshot]] (the wedge) | $495 flat | — |
| Email deliverability fix (SPF/DKIM/DMARC) | $500–$1,500 | [[Email Deliverability Playbook]] · [[Email Deliverability Engagement Template]] |
| M365/Workspace migration | $150/user, $2,000 minimum | [[M365 vs Google Workspace Strategy]] |
| Tenant security cleanup (existing M365/GW) | $1,500–$3,500 | — |
| Office network setup (new/renovated office) | $2,500–$8,000 + hardware at cost+15% | [[New Site Deployment]] |
| Server-to-cloud migration (kill the closet server) | $3,000–$10,000 | [[Server-to-Cloud Migration]] |
| Automation/AI mini-projects (existing clients only, year 1) | $1,500–$5,000 | — |

**Retainer clients get 15% off project work** — makes the retainer feel like a club.

Deliverability pricing note: the number of legitimate senders found in discovery is what moves the price from $500 toward $1,500 — the inventory step in [[Email Deliverability Playbook]] Phase 1.

Cash-flow rule: projects collect **50% up front**; retainers collect first-month-in-advance ([[Financial Model]]).
