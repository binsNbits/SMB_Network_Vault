# Launch Tool Stack

The **month-0, ≈$0–50/month** stack. Every client gets the **same stack** — standardization is the margin ([[Solo Operations and Boundaries]]). Upgrade order as revenue arrives: [[Tool Stack Upgrade Path]].

| Function | Tool | Cost |
|---|---|---|
| RMM (patching, monitoring, remote access) | Action1 | Free tier — first 100–200 endpoints, enough for ~8 clients |
| Remote support sessions | RustDesk (self-hosted or free tier) | Free |
| Ticketing/shared inbox | Zammad free tier (self-host) or shared mailbox + spreadsheet | Free |
| CRM / pipeline | HubSpot Free or a spreadsheet | Free |
| Password manager (yours + resell) | Bitwarden | Free / ~$4–6 per client org/mo |
| Documentation (per-client runbooks) | Obsidian/Notion — per [[Documentation Standards]] | Free |
| Quotes, invoices, payments | Wave or Zoho Invoice + Stripe/ACH | Free + payment fees |
| E-signature for agreements | Free tiers (e.g., Dropbox Sign 3/mo) | Free |
| Domain + M365 Business Basic for yourself | — | ~$15/mo |

## Rules

- **Target gross margin on retainers: 85%+** (labor is the cost). Tool COGS stays under **$15/user** against $95–135/user revenue ([[Retainer Tiers]]).
- Do **not** pre-buy per-seat tools with no seats to bill ([[Legal Insurance and Licensing]]).
- If a prospect insists on exotic tooling outside the standard stack: price it **40% higher or decline**.
