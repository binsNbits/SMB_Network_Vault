# Security Snapshot

The **$495 flat, fixed-scope, 1-week wedge engagement** that opens the [[Outreach Playbook|sales funnel]]. Deliverable: **a 2-page traffic-light report the owner can read in 3 minutes.**

## Why it works

Cheap enough to buy without a committee, concrete enough to say yes to, **always finds real problems** (every SMB fails at least 3 items), and the findings meeting *is* the retainer sales meeting ([[Retainer Tiers]]). Expect **40–60% of snapshots to convert** to retainers or paid remediation ([[Project Pricing]]).

**Free alternative when cold:** "15-minute cyber-insurance readiness check, no charge" — same conversation, lower barrier. Use it early when you need reps more than revenue.

## The checklist

Build the report template around the ~12–15 controls every insurance carrier's [[Cyber Insurance Questionnaire]] converges on — then one template works for every carrier:

- **MFA** on email, remote access, and admin accounts
- **EDR/managed antivirus** on all endpoints
- **Backups:** offline/immutable copies, tested restores → [[Backup Strategy]]
- **Patching** cadence and end-of-life software
- **Email security:** filtering + SPF/DKIM/DMARC → [[Email Authentication]], checked per [[Email Deliverability Playbook]] Phase 0
- **Password practices** / password manager
- **Privileged-account controls:** no shared admin, no daily-driver admin accounts, admin-account sprawl
- **Security awareness training** / phishing simulation
- **Incident response plan**, vendor/wire-transfer verification procedures → [[Security Incident Basics]]
- **Their last insurance questionnaire vs. reality** — ask for the actual application during the engagement

## Flow

List building → outreach → 15-min conversation → **Snapshot** → findings meeting → retainer proposal. Scripts and objection handling: [[Outreach Playbook]].
